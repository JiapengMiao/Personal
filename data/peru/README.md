# 秘鲁 2025 年白银贸易伙伴数据

`peru_silver_import_partners_2025.csv` 为 SUNAT Aduanas Perú 的年度“税则号/国家”查询结果，汇总 HS 7106 下有记录的两个 10 位子目：7106912000、7106920000。重量字段为 `net_weight_kg`；聚合后为 7,382.450 kg。

`peru_silver_export_confirmed_mirrors_2025.csv` 不是秘鲁海关完整出口表，而是目的国记录的“原产/伙伴为秘鲁”进口镜像。瑞士一行是 WSS 所列的 selected bullion imports，重量为毛重；美国、印度行为 HS7106 数量。三行的单位口径不完全一致，因而只能用于展示已确认的目的地样本，不能相加后解释为秘鲁全球出口总量或市场份额。

秘鲁 SUNAT 的同口径年度出口查询未返回 HS7106 记录；这一点与三国镜像流存在口径/申报归类差异，页面保留此边界说明，不作“出口为零”的结论。
