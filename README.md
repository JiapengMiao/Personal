# Project-002-白银数据网页可视化

## 项目概述
白银市场全景数据终端。原 Project-004、005、006 监测模块及 010 数据管道已统一并入本项目；Project-006 的客户池、客户分析与报告继续独立保留。项目使用 Vite、React、TypeScript 与 ECharts 构建交互式网页看板。

## 开始时间
2026-07-19

## 目录结构
- docs/    -> GitHub Pages 发布目录（仅网页构建产物，勿放对话记录等源文件）
- conversation_logs/ -> 对话记录与工作日志（仅本地保存，.gitignore 已排除，不上传 git）
- src/     -> 源代码；`collectors/` 为可选采集器，`update_all.py` 为统一更新入口，`pipelines/` 为辅助管道
- data/    -> 统一源数据：`wind/`、`shfe/`、`sge/`、`gfex/`、`monitoring/`
- output/  -> 更新运行记录、校验截图等产出物
- web/     -> 网页应用（Vite + React + TS + ECharts，public/data/ 为看板 JSON）

## 看板说明
- 启动：`cd web && npm run dev`（Vite，支持 `-- --port 7100 --host 127.0.0.1` 参数透传）
- 完整更新：在项目根目录运行 `python src/update_all.py`。它会检查源数据、生成全部 JSON、校验 TypeScript、构建网页并同步 `docs/`，但不会自动提交或推送 Git。
- 仅重建数据：`python src/update_all.py --data-only`；采集交易所或刷新 Wind 前先参阅 `src/collectors/README.md`。
- 看板区块：信号跑马灯 / Hero+紧张度仪表 / 01 市场脉搏 / 02 库存、递延费与铂钯仓单 / 03 持仓量与虚实比 / 04 COMEX / 05 上期所持仓 / 06 龙虎榜 / 07 贸易、现货基差、进出口盈亏与季节性 / 08 租赁利率 / 09 五项固定监测 / 10 趋势与结构 / 11 信号动态 / 12 十七项指标库+详情抽屉 / 方法论 / 页脚
- 交互：明暗双主题持久化、锚点滚动监听、count-up、tab 切换、图例显隐、dataZoom 缩放（全板块 slider+滚轮）、基差合约双下拉框+预设、指标筛选/搜索/排序、抽屉键盘导航

## 数据流
| 源 | 处理 | 产出 |
|----|------|------|
| `data/wind/白银所有数据.xlsx` 与 `data/wind/租赁利率/` | `src/build_dashboard_data.py` | 日频、持仓、季节性、租借利率、分钟线、基差、进出口盈亏等 JSON |
| Wind 主表《广期合约参数》《铂钯》+ `data/gfex/` | 合约纠错、重复行合并、交易日倒计时与仓单匹配 | `metal_virtual_ratio.json` / `pp_warehouse.json` |
| `data/shfe/ranking/` + `data/sge/` | 会员结构、非期货会员趋势、SHFE×SGE 持仓吨数对照 | `shfe_positioning.json` |
| `data/shfe/lhb/` | 汇总全历史龙虎榜 | `shfe_lhb.json` |
| `data/monitoring/` + Wind 库存列 | 监测底稿；指标16自动按 `LBMA+COMEX+上期+上金-SLV` 重算 | `monitoring.json` / `market.json` |
