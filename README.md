# Project-002-白银数据网页可视化

## 项目概述
白银市场全景数据终端：参考 Project-006 的深色金融终端网页设计，基于 Project-010 日度会议数据（Wind 主工作簿）并合并 Project-006 的五项监测与市场日频数据，构建的交互式可视化网页看板。

## 开始时间
2026-07-19

## 目录结构
- docs/    -> GitHub Pages 发布目录（仅网页构建产物，publish.bat 每次全量重建，勿放其他文件）
- conversation_logs/ -> 对话记录与工作日志（仅本地保存，.gitignore 已排除，不上传 git）
- src/     -> 源代码：build_dashboard_data.py（数据管道）、verify_dashboard.py / verify_drawer.py（截图验证）
- data/    -> 源数据副本（010源数据/、006源数据/、006设计参考/）
- output/  -> 产出物（verify/ 验证截图等）
- web/     -> 网页应用（Vite + React + TS + ECharts，public/data/ 为看板 JSON）

## 看板说明
- 启动：`cd web && npm run dev`（Vite，支持 `-- --port 7100 --host 127.0.0.1` 参数透传）
- 数据更新：`cd web && npm run build:data`（即 `python ../src/build_dashboard_data.py`，重跑后 JSON 即时生效，**无需重建前端**——数据为运行时 fetch，不打进 bundle）
- 看板区块：信号跑马灯 / Hero+紧张度仪表 / 01 五项固定监测 / 02 趋势与结构 / 03 市场脉搏（价格+LOF） / 04 递延费与库存（递延费+国内+海外+ETF） / 05 持仓量与虚实比 / 06 COMEX 头寸 / 07 基差与进出口盈亏（分钟级懒加载+合约选择） / 08 进口盈亏季节性 / 09 租借利率 / 10 信号动态 / 11 十七项指标库+详情抽屉 / 方法论 / 页脚
- 交互：明暗双主题持久化、锚点滚动监听、count-up、tab 切换、图例显隐、dataZoom 缩放（全板块 slider+滚轮）、基差合约双下拉框+预设、指标筛选/搜索/排序、抽屉键盘导航
