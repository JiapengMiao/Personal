### 6. dataZoom 滑块重置修复
- **根因**：dataZoom 配置中设置了 startValue（默认显示最近 250 天），每次 ECharts 重渲染时 startValue 被重新应用，覆盖用户拖拽位置
- **修复**：移除所有组件中 dataZoom 的 startValue 配置，默认显示全量数据，用户自行缩放

### 7. 布局重组：03 去库存 / 04 合并全部库存
- **用户反馈**：03 市场脉搏和 04 递延费都有国内库存图，重复了
- **修复**：
  - 03 市场脉搏：只保留价格面板 + 白银 LOF 基金面板（2 列）
  - 04 递延费与库存：递延费 + 国内库存（第一行）+ COMEX/注册仓单/PSLV + LBMA/英国ETF/SLV（第二行）
  - 删除 DailyOverseasSection 组件，合并入 DailySection
  - Market.tsx 删除 DomesticStocksPanel / OverseasStocksPanel / StockLines
  - App.tsx 移除 DailyOverseasSection 导入和使用

### 8. 持仓量/虚实比图对齐 010 参考
- **用户反馈**：虚实比图和 010 项目做出来的不一样
- **根因 1**：trading_calendar 在日历扩展之前构建，导致 OI 数据中超出主表日期范围的点（如 2026-07-20）searchsorted 返回错误位置
- **修复 1**：将 trading_calendar 构建移到日历扩展之后，从扩展后的 calendar 过滤周末
- **根因 2**：已到期合约曲线未延伸到 x=0
- **修复 2**：_contract_points 对已到期合约（expiry <= 最后数据日）自动补 x=0 端点
- **前端增强**：
  - 每条曲线末端添加 markPoint 标注（合约代码 + 最新值 + 距最后交易日 X 日）
  - x=0 处添加竖直虚线（markLine）
  - 图表右侧边距加宽到 120px 容纳标注文字
  - 图表高度从 300 增加到 340
